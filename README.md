# Верстка сервиса и доставки здорового питания Gastro-Chef. Пет-проект.

Вёрстка 7-ми страниц: 

    index.html
    about.html
    blog.html
    article.html       
    business-lunch.html
    gastro-shop.html
    gastro-shop_candies.html


Имеется: 
    
    адаптив.
    слайдеры.
    модальные окна.
    переключение по табам.
    аккардеоны.
    кастомные селекты.
    кастомные чекбоксы.
    валидация полей формы.
    инпут маска телефона.
    добавление товаров в заказ. 
    подсчёт стоимости всех товаров в заказе.


Зависимости: 

    jQuery
    jQuery-validation
    jQuery.maskedinput
    @fancyapps/fancybox
    slick-carousel
    
Запуск:

    1. npm i
    2. gulp
    3. gulp build - для билд сборки (опционально)
